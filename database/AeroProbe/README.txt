AeroProbe Pitot Calibrations
============================

This directory contains the Pitot Calibration Files (.pcf) provided
by AeroProbe containing calibration data for their six-port air data
probes.

The filenames contain the serial number of the Pitot tube in question
and the Mach number of the calibration run.

15239-1 M600-SN02 Port wingtip. Not calibrated.
15239-2 M600-SN02 Starboard wingtip. Not calibrated.
15239-3 M600-SN02 Main Pitot tube. Calibrated.
15239-4 Lost? Calibrated.
